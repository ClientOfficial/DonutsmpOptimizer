Welcome to my Minecraft DonutsmpOptimizer Project!
(ON TEST)

What is this?
A mod that improves Minecraft with new content, mechanics, and performance tweaks.

⚙️ How to install (Forge/Fabric)

Install Forge or Fabric.
Download the .jar file.
Install Fabric (latest version): https://fabricmc.net/
Install Fabric API: https://modrinth.com/mod/fabric-api
Move the DonutsmpOptimizer and Fabric API into your mods folder:
Press Windows + R
Type %appdata%
Go to .minecraft
If the mods folder doesn’t exist, create it
Launch Minecraft.
